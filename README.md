SYNRC 🌐 LDAP
=============
[![Hex pm](http://img.shields.io/hexpm/v/ldap.svg?style=flat)](https://hex.pm/packages/ldap)

compatible LDAP directory server with SQLITE, LMDB, MONGODB backends in 300 LOC of Elixir.
